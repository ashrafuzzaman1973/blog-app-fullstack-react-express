# 🚀 Full Stack Blog App

A simple full-stack blog application built using **React (Frontend)** and **Node.js + Express (Backend)** with file-based JSON storage.

---

## 🛠 Tech Stack

### Frontend:
- React.js
- Axios
- Tailwind CSS

### Backend:
- Node.js
- Express.js
- File System (JSON as database)

---

## ✨ Features

- 📝 Add new blog post
- 📄 View all blogs
- ❌ Delete blog post
- ⚡ Fast REST API
- 💾 JSON file-based storage (no database required)
